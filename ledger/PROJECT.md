# RomanGrid

## Overview
Music theory app for chord charts. Users enter chords and a key, and the app converts them to Roman numeral notation for on-the-fly transposition. Designed for musicians who need to play songs in any key. Built as a native iOS/macOS app in SwiftUI (Xcode project).

## Key Features
- Chord entry grid editor
- Roman numeral conversion from absolute chords + key
- Drag-to-extend chord events across cells
- Auto quality selection based on scale degree and mode
- Support for all modes (Ionian, Dorian, Phrygian, etc.)
